# Odin Recipes

## Introduction

This project is part of The Odin Project and focuses on learning the basics of HTML and havig a hand around Git.

The goal of this project is to build a simple recipe website using HTML while also learning how to use Git and GitHub to manage files and track changes.

By completing this project, I will practice creating web pages, linking pages together, and organizing a small website project.

## Skills demonstrated

By completing this project, I will have demonstrated the ability to:

- Use Git and GitHub for version control
- Create and structure basic HTML pages
- Use links to connect multiple pages
- Organize a simple website project
- Commit and push changes using the macos terminal
